# Aftra -- Color System

> Source: Brand Identity Guidelines v20250428, Section 2.0

## Primary Palette

Four core colors. Moody Blue and Purple Heart each have a 6-step scale (050 tint through 500 shade). Neutrals have 3-6 values.

### Moody Blue (primary -- bold statements, CTAs)

| **300** | **`#5A4DDE`** | **Primary brand color** |
| 400 | `#342B99` | |
| 500 | `#211D49` | Shade / dark accents |

## Accent Colors

Four accents for status, data viz, and emphasis. Each has a 6-step scale (050-500).

| Name | Key Hex (200/300) | Tint 050 | Role |
|---|---|---|---|
| Green Circuit | `#16A374` | `#D7FAEE` | Success, positive |
| Gold Brandy | `#FACC15` | `#FEFBDA` | Warning, highlight |
| Red Tabasco | `#DA481C` | `#FEEADA` | Error, critical |
| Violet Electric | `#AC1AD0` | `#F6DBFF` | Info, special |
