# Aftra -- Typography

> Source: Brand Identity Guidelines v20250428, Section 3.0

## Primary Typeface: Poppins

Used for **section headers, body text, and labels**. Geometric sans-serif from Google Fonts.

### Weights in use

| Weight | CSS `font-weight` | Typical usage |
|---|---|---|
| Extra Bold | 800 | Hero headlines, brand statements |
| Bold | 700 | Section headings |
| Semi Bold | 600 | Subheadings, emphasis |
| Medium | 500 | Nav items, labels, UI text |
| Regular | 400 | Body copy |
| Light | 300 | Supporting text, captions |

## Secondary Typeface: Roboto Slab

Slab-serif companion. Used for **display / editorial contexts** -- likely large headlines on the brand guide cover and section dividers. Not typically used in product UI.

## Implementation Notes

Both fonts are available on Google Fonts:
- `https://fonts.google.com/specimen/Poppins`
- `https://fonts.google.com/specimen/Roboto+Slab`

For web, load via Google Fonts or self-host. Suggested import for the weights we need:

```
Poppins:wght@300;400;500;600;700;800
Roboto+Slab:wght@400;700
```
